Escandalo! DJ Request App

A real-time song request and shoutout app built for DJ Hexx's Escandalo! sets.

Features


🎵 Guests scan a QR code to submit song requests or shoutouts
⚡ Real-time dashboard powered by Firebase Firestore
💸 Built-in tip link (CashApp)
📱 Instagram follow link on confirmation
🧹 One-click clear all requests for the DJ
🎨 Custom styled with Tailwind CSS


Tech Stack


React + Vite
React Router
Firebase Firestore (real-time database)
Tailwind CSS


Pages


/request — Guest-facing form to submit song requests or shoutouts
/dashboard — DJ-facing live dashboard showing incoming requests in real time
